# F550 Hexacopter 5L Sprayer Conversion

## 1. Propulsion Upgrade (5L Payload Specs)
To lift 5kg of liquid + 3kg drone weight (8kg AUW), the F550 must transition to 6S power.
* **Motors:** 6x [Xoar Titan T4006 350kv](http://googleusercontent.com/shopping_content/3_link).
* **ESCs:** 6x [Hobbywing Skywalker 60A V2](http://googleusercontent.com/shopping_content/4_link).
* **Props:** 3 Pairs [Tarot 15x5.5 Carbon Fiber](http://googleusercontent.com/shopping_content/5_link).
* **Arms:** AerialMob Arm Extension Set (16mm Carbon Booms).

## 2. RTK & Navigation
* **Rover:** [Here3+ RTK GNSS](http://googleusercontent.com/shopping_content/6_link) (DroneCAN).
* **Base Station:** Holybro H-RTK F9P Base (Connects to Getac T800).

## 3. Power Configuration
* **Series Setup:** Use a **Series Y-Harness** to connect 2x 3S LiPo batteries (identical mAh) to create a single 6S source.
* **Current Handling:** Upgrade PDB (Power Distribution Board) to handle 150A+ total burst.

## 4. Spray System Integration
* **Tank:** 5L Under-slung graduated tank.
* **Pump:** 12V Diaphragm pump controlled via PWM Relay on Pixhawk AUX 1.
* **Nozzles:** Dual anti-drip nozzles mounted on arm extensions.

## 5. Estimated Upgrade Costs
* **Propulsion (Motors/ESC/Props):** ~$665.
* **Frame Extensions:** ~$120.
* **RTK Rover/Base Kit:** ~$650.
* **Total Investment:** ~$1,435.
